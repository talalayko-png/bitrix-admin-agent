# moysklad-app (задел)

Зарезервировано под встроенное приложение/виджет для интерфейса МойСклад
(аналог `bitrix24-app`, но со стороны МС): показ связанной сделки Bitrix24,
статуса синхронизации и действий прямо в карточке заказа МойСклад.

Сейчас это **только задел** — реализация запланирована, см.
[../../ROADMAP.md](../../ROADMAP.md) (раздел «Среднесрочное»).

Вся бизнес-логика останется на бэкенде; здесь будет лишь тонкий UI поверх
Admin API, как и в `bitrix24-app`.

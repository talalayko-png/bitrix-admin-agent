# bitrix24-app (iframe-виджет)

Тонкий виджет для карточки смарт-процесса/сделки Bitrix24. Вся логика — на
бэкенде; виджет только показывает статус синхронизации и даёт кнопки
**Синхронизировать / Dry-run / Повторить / Подтвердить**, дёргая Admin API.

## Запуск (демо)

Это статическая страница без сборки. Откройте её любым статическим сервером,
например:

```bash
cd frontend/bitrix24-app
python3 -m http.server 5174
# затем http://localhost:5174
```

В разделе «настройки» укажите `API base` (например `http://localhost:8000`) и
`Admin token` — они сохранятся в localStorage.

> Добавьте origin виджета (например `http://localhost:5174`) в `CORS_ORIGINS`
> бэкенда, иначе браузер заблокирует запросы.

## Встраивание в Bitrix24

При размещении как placement-виджет id сделки читается автоматически из
`BX24.placement.info()`. В проде авторизацию следует брать из Bitrix24, а не
вводить admin-токен вручную (см. ROADMAP — OAuth-установка).

# Roadmap

## ✅ MVP (текущая сборка)

- Скелет backend (FastAPI) + воркер (RQ) + Redis + Postgres в docker-compose.
- Входящие вебхуки Б24/МС с проверкой HMAC.
- Очередь операций: ретраи, экспоненциальный backoff, Redis-локи, идемпотентность.
- Workflow-движок + пример `deal_to_order`.
- Approval gate, snapshots, журнал операций.
- Admin API + рабочая admin-панель (React) + тонкий bitrix24-app (iframe).
- Тесты на моках (без сети), три предохранителя исходящих вызовов.

## 🔜 Ближайшее

- **Реальные коннекторы** Bitrix24 и МойСклад (за предохранителями), запись
  и интеграционные тесты против песочниц.
- **Alembic-миграции** вместо `create_all`.
- Больше workflow: товары/остатки, контрагенты, отгрузки, оплаты.
- Пагинация и расширенные фильтры в Admin API/панели.
- Метрики (Prometheus) и трассировка.

## 🧭 Среднесрочное

- **moysklad-app**: полноценный встроенный виджет МойСклад (сейчас только задел
  в `frontend/moysklad-app/`).
- OAuth-установка приложений в маркетплейсах Б24/МС.
- Секреты в Vault/KMS, ротация токенов.
- RBAC для admin-панели (роли наблюдатель/оператор/админ).
- Подключение **AI-ассистента** (сейчас плейсхолдер `assistant`): объяснение
  ошибок синхронизации, подсказки по маппингу, генерация правил workflow.

## 🌱 Идеи

- Конструктор workflow в UI (визуальный редактор правил).
- Песочница «прогон события» с пошаговой визуализацией.
- Экспорт/импорт конфигурации интеграции.
